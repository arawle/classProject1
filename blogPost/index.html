Math.js Tutorial

What is Math.js?

Math.js is a JavaScript and Node.js library which aims to add more functionality to the built in JavaScript Math library. Math.js allows for the use of complex and imaginary numbers, arrays, and matrices.

How to install Math.js

You can install directly with npm using 
npm install mathjs
or by linking to the CDN page

Where to implement Math.js

The easiest way to use Math.js in your web applications is to put the CDN script in the html head section 
<html>
<span>
<title> Example </title>
<script type='text/javascript' src='http://cdnjs.cloudflare.com/ajax/libs/mathjs/2.1.1/math.js'></script>
...

What can Math.js do?

Evaluation of Expressions

math.eval( ) allows for the user to enter a mathematical equation and for the computer to evaluate it. Math.js allows evaluation of ( ), [ ], ^ , ƒ, in addition to the characters that js can already evaluate like + and -.

Complex Numbers

math.js has a number of features that allow it to deal with complex numbers like i.

Matrices

math.js has many built in functions for manipulating matrices, including many which are helpful using vector calculus.
  • Cross Product
    math.cross(x,y) finds the cross product of two three-dimensional vectors. The cross product of two vectors is a vector perpendicular to A x B.

  • Dot Product
    math.dot(x,y) find the dot product of two vectors of equal length. The dot product is the projection of one vector onto another. If two vectors are perpendicular to eachother, the dot product is zero.

  • Trace
    math.trace(x) is the trace of a square array or matrix. The trace is the sum of the main diagonal of x. Finding the trace is significant if the matrix is a geometric projection in that the trace is the dimension of the space that it is being projected onto (common in Ordinary Differential Equations).

  • Determinant
    math.det(x) is the determinant of a matrix. Determinants are especially useful in a subset of Topilogical Manifolds called knot theory. It is used in determining the mod coloring and Alexander Polynomial of a knot.

Probability and Statistics

math.js has many probability and statistics functions, and also can normalize data. A few of the more common statistics equations are on my calculator (probability coming soon).

Calculator Instructions

  • The default setting for sin/cos/tan is radians. Use the ° button if you would like to calculate an angle in degrees.
  • To graph use the f(x)= button, plus the equation that you want to graph, then hit = . Once you have graphed, use your mouse to move around the coordinates as well as zoom in and out
  • Matrix, Statistics, and Probability should not be clicked until you have typed your full equation in.
