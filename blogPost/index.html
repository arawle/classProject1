Math.js Tutorial

What is Math.js?

Math.js is a JavaScript and Node.js library which aims to add more functionality to the built in JavaScript Math library. Math.js allows for the use of complex and imaginary numbers, arrays, and matrices.

Why use Math.js?

How to install Math.js

You can install directly with npm using 
npm install mathjs
or by linking to the CDN page

Where to implement Math.js

The easiest way to use Math.js in your web applications is to put the CDN script in the html head section 
<html>
<span>
<title> Example </title>
<script type='text/javascript' src='http://cdnjs.cloudflare.com/ajax/libs/mathjs/2.1.1/math.js'></script>
...

What can Math.js do?

math.eval( )

math.eval( ) allows for the user to enter a mathematical equation and for the computer to evaluate it. Math.js allows evaluation of ( ), [ ], ^ , ƒ, in addition to the characters that js can already evaluate like + and -.
